This plugin provides log your post activities.

== Description ==

Using WP Posts Log you can track publish, edit, trash, draft, and schedule statuses of your posts. You can also filter logs by user name and find out when changed the post status.


= Translations =

* English (en_US), built-in
* Turkish (tr_TR), native support

== Installation ==

* Download the repository to your plugins directory.
* Visit your plugins page and activate the plugin.
